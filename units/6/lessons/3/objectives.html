By the end of this lesson, students will be able to:

  • Describe how multicore and multithreaded CPUs work, and their use cases and benefits, and how OpenMP directives work.
  • Find the compute intensive parts of programs and where to place OpenMP directives.
  • Measure performance improvements or slowdowns from code changes.
  • Measure problem complexity and performance scaling.
  • Avoid bad practices in OpenMP programming that could lead to potential performance issues.
  • Accelerate scientific application kernels using OpenMP.
