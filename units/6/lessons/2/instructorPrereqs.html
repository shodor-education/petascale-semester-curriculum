Before teaching this lesson, the instructor should have:

  • Access to a supercomputer and be able to provide access for the students taking the course. This should include contacts that the students can reach out to for login issues, issues with running jobs/accessing the supercomputer, etc.
  • Understanding of distributed memory parallelism and shared memory parallelism and how to parallelize basic applications with each.
  • The ability to explain why a hybrid OpenMP+MPI program could be useful and in what context it would be most beneficial.
