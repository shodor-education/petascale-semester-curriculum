Prior to teaching this module, the instructor should understand and be able to do the following:

  1. Have access to a supercomputer and be able to provide access for the students taking the course. This should include contacts that the students can reach out to for login issues, issues with running jobs/accessing the supercomputer, etc.
  2. Understanding of distributed memory parallelism and shared memory parallelism and how to parallelize basic applications with each
  3. Be able to explain why a hybrid OpenMP+MPI module could be useful and in what context it would be most beneficial
