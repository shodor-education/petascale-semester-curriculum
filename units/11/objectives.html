At the end of this lesson students will be able to:

  1. Install the PLUTO software package to solve problems in astrophysical fluid dynamics.
  2. Adapt and create the initialization code to solve specific problems.
  3. Compile, schedule and run jobs on a parallel computing cluster.
  4. Visualize simulation output using the VisIt software.
  5. Examine code performance and scalability of 2D and 3D simulations at a range of mesh resolutions, optimize.
  6. Advanced goal: use and optimize Adaptive Mesh Refinement
