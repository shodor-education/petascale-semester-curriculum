By the end of this lesson, students will be able to:

  • Install the PLUTO software package to solve problems in astrophysical fluid dynamics.
  • Adapt and create the initialization code to solve specific problems.
  • Compile, schedule and run jobs on a parallel computing cluster.
  • Visualize simulation output using the VisIt software.
  • Examine code performance and scalability of 2D and 3D simulations at a range of mesh resolutions, optimize.
