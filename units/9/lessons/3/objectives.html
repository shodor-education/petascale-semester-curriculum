By the end of this lesson, students will be able to:

  • Describe
    • A cache memory
    • Why computer systems include cache memories
    • Where the cache memory is located in a computer system
    • The technology a cache memory uses
    • The principles that make a cache memory work
  • Write a C program to
    • Accept/verify command line arguments
    • Allocate a linear array of elements of any data type
    • Measure and display the time it takes to fill the array
  • Write a basic BASH shell script to execute a program with different command line arguments
