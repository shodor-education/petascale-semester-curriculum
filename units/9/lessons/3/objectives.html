This lesson will equip the students to:

  1. Describe
    1. A cache memory
    2. Why computer systems include cache memories
    3. Where the cache memory is located in a computer system
    4. The technology a cache memory uses
    5. The principles that make a cache memory work
  2. Write a C program to
    1. Accept/verify command line arguments
    2. Allocate a linear array of elements of any data type
    3. Measure and display the time it takes to fill the array
  3. Write a basic BASH shell script to execute a program with different command line arguments
