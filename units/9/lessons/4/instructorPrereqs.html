In terms of knowledge and background, the instructor prerequisites are the same as the student ones.

In preparation for using the lesson, the instructor should test the given code on their platform to ensure that it shows the effect of false sharing. In particular, they may want to customize the size of image being given so that the measured part of the running time is stable for both the false sharing (i.e. small multiplier) and non-false sharing (i.e. large multiplier) versions of the code. Images of any size can be created by running the mandelbrot.c code from the “Race conditions” lesson of unit 4; simply modify the variables numRows and numCols (ideally keeping them the same).
