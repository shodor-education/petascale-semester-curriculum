At the completion of this unit, students should be able to...

  • install and configure CUDA and run example programs
  • compile and run a piece of prewritten CUDA code
  • decide when to utilize a GPU vs a CPU
  • describe the architectural difference between CPU and GPU
  • allocate memory on host and device and copy between
  • write and call a CUDA kernel
  • remap a loop index to blocks and threads
  • demonstrate the impact of branching on GPGPU efficiency
  • profile and debug CUDA code
  • utilize graphics interop capabilities
  • call CUDA from languages other than C++
