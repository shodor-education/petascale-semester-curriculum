By the end of this unit, students will be able to:

  • Know OpenMP concepts, language definitions, and directives.
  • Apply OpenMP parallelization concepts to accelerate stencil/convolution code.
  • Know how to accelerate code on the GPU using OpenMP 4.5 new pragma target.
  • Understand when using OpenMP parallelization would be useful and when it would not be useful.
  • Understand the practices and applications of OpenMP.
