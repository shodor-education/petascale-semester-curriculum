Before starting this lesson, students should have:

  • Basic understanding of OpenMP "parallel", "parallel for", and "for" pragmas.
    • This likely means that Lessons 4.3 (Introduction to OpenMP), 4.6 (When Should You Use OpenMP?), and 4.10 (Ensemble Based Simulated Annealing in OpenMP) — which has some discussion of thread safe functions) should be covered first.
  • Understanding of shared versus private variables.
  • Understanding of what it means for a function to be threadsafe (see Lesson 4.10 (Ensemble Based Simulated Annealing in OpenMP)).
  • Understanding of C++, compiler use, how to use multiprocessor machines (See Unit 3 for the various skills required).
  • If running on Blue Waters, how to run jobs in the queue system.
