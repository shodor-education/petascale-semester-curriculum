This activity is geared towards students who have begun learning OpenMP and are moving towards examples that will allow them to put skills in practice.

Students should have prior knowledge of

  • Programming in C
  • Max/Min Problems
  • OpenMP
    • Concerns over race conditions and shared memory
    • Definition of concurrency and loop carried dependency
    • OpenMP format for parallel for loop with private clauses

Additionally, it will be useful for instructors to review max/min problems in general, the problem of global versus local minimization, the Metropolis-Hastings algorithms, and the use of the simulated annealing problem in practice in order to be able to answer student questions.
