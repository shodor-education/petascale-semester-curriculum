Before teaching this lesson, the instructor should have:

  • Understanding of "for" loops and variable scoping in C.
  • Understanding of the idea of parallel execution using threads.
  • Familiarity with the OpenMP pragmas used ("parallel for" with "private" and "reduce"), as well as the concepts of race conditions and reductions.
  • The ability to explain the Mandelbrot set and how the given code corresponds to the definition; this would be potentially helpful, though it is not strictly necessary.
