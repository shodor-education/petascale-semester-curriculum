Before teaching this lesson, the instructor should have:

  • Experience in programming in the C language, knowledge of OpenMP, and how to utilize a parallel computer system.
  • Understanding and experience with the shared memory concepts of OpenMP, the practices, and the applications.
  • Working program examples for showing how the OpenMP of applications and practices work.
  • Access to a multicore computer with C and OpenMP installed.
  • Provided every student in the course access to the multicore computer with C and OpenMP installed.
