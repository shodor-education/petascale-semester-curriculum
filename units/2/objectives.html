By the end of this unit, students will be able to:

  • Understand and be able to identify (1) task parallelism and (2) data parallelism within a problem.
  • Understand and be able to describe (1) shared memory and (2) distributed memory approaches.
  • Describe parallel programming patterns to achieve peak parallel performance.
  • Describe the sources of parallel overhead, performance metrics and scalability of a parallel system.
  • "Think in parallel" having examined several parallel problems including vector/matrix addition, sorting numbers, and random number generation.
