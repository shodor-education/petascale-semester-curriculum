1) Students will understand and be able to identify (1) task parallelism and (2) data parallelism within a problem.

2) Students will understand and be able to describe (1) shared memory and (2) distributed memory approaches.

3) Students will learn parallel programming patterns to achieve peak parallel performance.

4) Student will be able to describe the sources of parallel overhead, performance metrics and scalability of the parallel system

5) Students will learn to “think in parallel” by examining several parallel problems including vector/matrix addition, sorting numbers, and random number generation.
