Before teaching this lesson the instructor should be familiar with the following topics:

1. Shared memory

2. Distributed memory

3. Basic networking and network topologies common in high-performance computing (point-to-point, ring, crossbar, toroidal mesh, 2D planar mesh, and 3D planar mesh)

4. Basic computer interconnect performance and resiliency metrics including bisection bandwidth and how to model traffic flow/identify bottlenecks
