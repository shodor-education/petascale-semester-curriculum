Before starting this lesson students should have:

  • A basic understanding of the stored program concept and that problem data must be stored in memory.
  • A basic understanding of networking — specifically that two or more physical separate computer systems can be connected together via a communications network (e.g. understanding that the Internet connects many computers together is sufficient).
  • A basic understanding of how message traffic can flow in a network — any knowledge of basic flow control (e.g., water or current) is sufficient.
