Before starting this module, students should have:

  1. Basic familiarity with using a computer organization and architecture
  2. Basic knowledge of supercomputers.
  3. Beginner level knowledge of any programming paradigm (C, C++, Java...)
  4. A basic understanding of the stored program concept and that problem data must be stored in memory.
  5. A basic understanding of networking – specifically that two or more physical separate computer systems can be connected together via a communications network (e.g., understanding that the Internet connections many computers together is sufficient).
  6. A basic understanding of how message traffic can flow in a network – any knowledge of basic flow control (e.g., water or current) is sufficient.
