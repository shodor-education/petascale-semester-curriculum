Before teaching this lesson, the instructor should have:

  1. Access to a compute node with GPU or workstation with GPU accelerators for testing and verifying examples and exercise.
  2. Understanding of GPU architecture and what kind of scientific applications could benefit from them.
  3. Check with the PGI community for any new releases and changes in OpenACC and make necessary modifications to the examples and exercises.
  4. Familiarity with C base compilers
