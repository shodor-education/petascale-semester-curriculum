• Instructors and students must have access to a cluster/supercomputer with GPU nodes or at least a workstation with GPU accelerator(s).
  • Most supercomputers will have the PGI toolkit installed. If not, a request for installation of new tools must be submitted. If using a workstation, make sure the PGI community edition of the toolkit is installed.
  • Emacs, Vim, or any other code editors that support Fortran and C compilers on the workstation.
  • Example and exercises codes from this lesson.

An interactive job run system is very desirable, i.e., not a batch job queue submission system.
