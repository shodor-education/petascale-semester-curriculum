Before teaching this lesson, the instructor should have:

  • Access to a compute node with GPU or workstation with GPU accelerators for testing and verifying examples and exercises using OpenACC.
  • Understanding of GPU architecture and what kind of scientific applications could benefit from them.
  • Checked with the PGI community for any new releases and changes in OpenACC and made necessary modifications to the examples and exercises.
  • Familiarity with Fortran and C base compilers.
  • Understanding of how to use ImageMagik.
