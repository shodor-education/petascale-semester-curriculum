By the end of this lesson, students will be able to:

  • Know how to evaluate a loop to see if it is a candidate for parallelization using a vector architecture.
  • Gain experience predicting performance of a loop on a vector architecture.
  • Understand how to employ Foster's methodology (see: Designing and Building Parallel Programs, by Ian Foster, available at: https://www.mcs.anl.gov/~itf/dbpp/ ) for designing parallel programs for a given architecture.
