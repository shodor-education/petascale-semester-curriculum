Before starting this lesson students should have:

  1. A basic understanding of the stored program concept and that problem data must be stored in memory.
  2. Have seen a GPU (SIMD), and CPU (SISD) before.
  3. A basic understanding of how programs are executed on a computer – instructions are used to tell the CPU what to do and data are processed.
  4. A basic understanding of caches and how they are used in computing. Understanding that they are a small and fast memory is sufficient.
  5. A basic understanding of data access and memory access times.
  6. A basic understanding of how to create a plan to solve a given problem for the Foster’s methodology material– not necessarily how to design software, but if they have written any code that is fine.
  7. A basic understanding of loops.
  8. A basic understanding of Foster’s methodology. Material from lesson 3.1 is sufficient for this item.
  9. A basic understanding of Flynn’s taxonomy. Material from lesson 3.1 is sufficient for this item.
