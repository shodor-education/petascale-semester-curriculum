Module 3.2 – Instructor Prerequisites



Before teaching this module the instructor should be familiar with the following topics:


  1. Flynn’s taxonomy
  2. Vector architectures
  3. Loop unrolling
  4. Matrix multiplication
  5. Foster’s methodology (see: see: Designing and Building Parallel Programs, by Ian Foster, available at: https://www.mcs.anl.gov/~itf/dbpp/ ) for designing parallel programs
