By the end of this lesson, students will be able to:

  • Know how to login to Compute Canada's clusters such as Cedar.
  • Recall different filesystems that could be used for running their parallel applications.
  • Learn how to use slurm scheduler for running jobs through script.
  • Learn how to request interactive nodes for testing and practice.
  • Explore different ways of running OpenMP, MPI, and OpenACC codes on Cedar Cluster computer using Slurm Scheduler.
