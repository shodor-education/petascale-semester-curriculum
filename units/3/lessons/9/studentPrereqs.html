Before starting this lesson, students should have:

  • Knowledge of common Linux commands:
    • ls — list contents of the present working directory
    • cd — change to identified directory
    • pwd — show present working directory
    • cp — copy source to target
    • mv — move source to target
  • Ability to open a terminal text editor:
    • emacs
    • vi
  • Knowledge how to write a job file and submit it to the scheduler.
