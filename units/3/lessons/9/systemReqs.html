Most of the example codes could be compiled and run under any Linux system. In order to follow the materials in this module, SLURM job scheduler and PGI compiler, and OpenMPI version 3.1.2 or higher is mandatory

  1. Good to have:
    1. Access to Cedar or Graham Cluster Computers through Compute Canada
    2. GPU device
    3. Intel Compilers
  2. Must have:
    1. unix/linux utilities: tar, scp, make, ssh, command line terminal,
    2. PGI C compiler 19.4+, Intel C compiler 2018.3+, OpenMPI 3.1.2+, GNU C compiler
    3. Access to at least 2 node cluster with multiple cores per node
