For students to be able to successful understand and complete this lesson students should have the following prerequisites:

  • Be familiar with the Unix/Linux environment (Basic commands such as: mkdir, cd, ls, cp, pwd, etc ..)
  • Have remote access to a supercomputer systems
  • Be able to edit, compile, and run a C/C++/Java or other high level programming language source code program
  • Be able to write and run a script written mostly in perl
