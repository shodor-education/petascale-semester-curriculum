Before teaching this lesson, the instructor should have:

  • Familiarity with job submission using SLURM.
  • Helped the students secure accounts in XSEDE.
  • Obtained an educational allocation in XSEDE and requested access to Stampede2. As a guideline, the instructor should expect to request 500 SUs per student. Educational allocations are typically awarded in two weeks.
  • Familiarity launching jobs using ibrun and srun.
  • Familiarity with the LUSTRE file system including access to scratch file systems as well as the latter policies.
