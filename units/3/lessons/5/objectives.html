By the end of this lesson, students will be able to:

  • Know how to login to submit and run jobs using SLURM.
  • Recall different filesystems that could be used for running their parallel applications.
  • Know how to use slurm scheduler for running jobs through script.
  • Recall different ways of running OpenMP, MPI, and OpenACC codes.
  • Recall the basics of NUMA control.
