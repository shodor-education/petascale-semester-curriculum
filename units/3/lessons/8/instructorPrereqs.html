The following prerequisites must be satisfied before proceeding with the module:

  1. The instructor must be familiar with job submission using SLURM.
  2. The instructor should help the students secure accounts in XSEDE.
  3. The instructor should obtain an educational allocation in XSEDE and request access to Stampede2. As a guideline the instructor should expect to request 500 SUs per student. Educational allocations are typically awarded in two weeks.
  4. The instructor should be familiar launching jobs using ibrun and srun.
  5. The instructor should be familiar with the LUSTRE file system including access to scratch file systems as well as the latter policies.
