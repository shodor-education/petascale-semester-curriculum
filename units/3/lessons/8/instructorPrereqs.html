Before teaching this lesson, the instructor should have:

  • Proficiency in managing Unix/Linux environment.
    • All basic linux commands (mkdir, rmdir, cd, ls, touch, nano, pico, cp, mv, rm, ..).
    • A few advanced linux commands (w, whereis, man, emacs, vim, ps, df, du, diff,..).
  • Accessed and managed remotely a supercomputer (ssh, putty, ...).
  • Loaded a software package, compiled a source code.
  • Edited and run a job script.
  • Read error and output files.
