By the end of this lesson, students will be able to:

  • Perform file transfers using SFTP clients.
  • Perform file transfers using the SCP command in a SSH terminal session.
  • Copy/paste code from their machine to an empty file created on a remote cluster using the terminal.
  • Use the 'dos2unix' command to make files created in Windows (and 'mac2unix' command to make files created in Mac) machines conform to the Linux standard.
