Prerequisite Assessment

  • Submit screenshots of Linux CLI environment.

Assessment 1: Understand the REPL concepts

  • Display messages to terminal using echo

Assessment 2: Understand directory tree structures and relative/absolute path

  • Given a directory tree, require students to provide/select the correct command to change from one directory to another.
  • Given a directory and a sequence of command, ask students to provide/select the correct outcome of pwd.

Assessment 3: Creating/editing files

  • Ask students to create a file with some contents.
  • Ask students to download an existing file and modify its contents.

Assessment 4: Deleting files/directories

  • Ask students to download a zipped file containing some directory structures, then to delete certain directories/files.
