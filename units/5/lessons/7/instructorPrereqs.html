Before teaching this lesson, the instructor should be proficient in:

  • Managing a Unix/Linux environment
    • All basic Linux commands (mkdir, rmdir, cd, ls, touch, nano, pico, cp, mv, rm, ...)
    • A few advanced Linux commands (who, whereis, man, emacs, vim, ps, df, du, diff, ...)
  • Accessing and managing a supercomputer remotely (ssh, PuTTY, ...)
  • Loading a software package (in this case OpenMPI or equivalent MPI package, compiling a source code.
  • Editing and running a job script.
  • Reading / analyzing error and output files
  • Debugging and restarting processes if necessary
