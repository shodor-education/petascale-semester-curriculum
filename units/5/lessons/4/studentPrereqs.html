Before starting this lesson, students should have solid understanding of:

  1. Parallel programming concepts
  2. Programming in C/C++
  3. Distributed Memory architecture and concepts
  4. Introduction to MPI
  5. MPI Point-to-Point Communication and Collective Communication routines and algorithms.
  6. Able to compile and run code on large cluster/supercomputer
  7. Difference between CPU time and wall clock times
