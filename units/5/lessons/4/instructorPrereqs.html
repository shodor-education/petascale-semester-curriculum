Before teaching this lesson, the instructor should have:

  • Access to a compute node cluster/supercomputer or workstation with the OpenMPI library installed.
  • Understanding of cluster computers/supercomputer architectures and what kind of scientific applications could benefit from them.
  • Familiarity with the Top500 List of world supercomputers.
  • Experience programming in C/C++ as well as the MPI library
  • Understanding of problem decomposition and distributed memory parallelism concepts as well as understanding of different MPI modes for communication such as blocking vs. non-blocking and collective communication algorithms.
  • Checked with the Open MPI Organization/Community for any new releases and changes to the OpenMPI library and make necessary modifications to the examples and exercises.
