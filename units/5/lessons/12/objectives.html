By the end of this lesson, students will:

  • Knowledge of what is a network switch connected computer cluster, how it works, and its use cases.
  • How to find the compute intensive parts of programs and the associated memory spaces.
    • Understand the methods for decomposition of program memory arrays into partitions.
    • How to copy data between CPU <=> CPU, how to distribute data.
  • Learn by example how to avoid bad practices in MPI programming that could lead to potential performance issues
  • Hands-on examples and exercises for accelerating scientific applications using domain memory decomposition across many compute nodes.
