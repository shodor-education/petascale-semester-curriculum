By the end of this lesson, students will be able to:

  • Recall what a network switch connected computer cluster is, how it works, and its use cases.
  • Find the compute intensive parts of programs and the associated memory spaces.
    • Understand the methods for decomposition of program memory arrays into partitions.
    • Copy data between CPU <=> CPU and distribute data.
  • Avoid bad practices in MPI programming that could lead to potential performance issues.
  • Accelerate scientific applications using domain memory decomposition.
