Before teaching this lesson, the instructor should have:

  • Knowledge about distributed memory concepts and MPI.
  • Experienced how distributed memory concepts and MPI are implemented for writing and running parallel programs, including how to change a program from a single core program to a multicore program and how to fine tune the parallel program to make it scalable.
  • Access to, and know how to use, a parallel computer, including how to transfer files from the local to the parallel computer and vice versa, and how to submit jobs to the parallel computer.
  • Provided the students access to a parallel computer.
  • Demonstrated how to use a Unix/Linux machine.
