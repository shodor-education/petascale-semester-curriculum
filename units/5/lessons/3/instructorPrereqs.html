1. The instructor needs to know about the distributed memory concepts and MPI.
  2. The instructor has experienced how the distributed memory concepts and MPI are implemented for writing and running parallel programs including how to change a program from a single core program to a multicore program, how to fine tune the parallel program to make it scalable, etc.
  3. The instructor has access to and knows how to use a parallel computer including how to transfer files from the local to the parallel computer and vice versa, how to submit jobs to the parallel computer.
  4. The instructor provides the students access to a parallel computer.
  5. The instructor has demonstrated how to use a Unix/Linux machine.
