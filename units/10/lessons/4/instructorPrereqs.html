In addition to the prerequisites of Lesson 10.3 listed below:

  • Be proficient in the Unix/Linux environment
  • Have a clear command of programming in basic programming languages such as C/C++/Java/ python and more if needed.
  • Be able to edit, compile, run and debug source codes written in the languages listed above
  • Be Familiar with parallel programming API such as MPI, OpenMP, CUDA, etc..
  • Be able to edit, compile, run, and debug code written in the above listed parallel programming API
  • Mostly be able to use a supercomputer work load manager and scheduler such as SLURM, TORQUE or any other well know beside these 2.

The instructor should be able to perform these complementary prerequisites:

  • Modify, Debug, and optimize a code written with parallel programming API such as MPI, OpenMP, and other.
  • To assist students locate very quickly students errors or problems and help them to stay on speed.
