Before teaching this lesson, the instructor should have:

  • Proficiency in the Unix/Linux environment.
  • A clear command of programming in basic programming languages such as C/C++/Java/Python and more if needed.
  • The ability to edit, compile, run and debug source codes written in the languages listed above.
  • Familiarity with a parallel programming API such as MPI, OpenMP, CUDA, etc.
  • Ability to edit, compile, run, and debug code written in the above listed parallel programming API.
  • Ability to use a supercomputer workload manager and scheduler such as SLURM, TORQUE or any other well known workload managers.
