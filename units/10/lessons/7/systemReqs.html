• Instructors and students must have access to a cluster/supercomputer with/without GPU nodes or at least a workstation/laptop.
  • Most supercomputers will have PGI toolkit installed. If not, a request for installation of new tools must be submitted. If using a workstation, make sure the PGI community edition of the toolkit is installed.
  • Emacs, Vim, or any other code editors that supports Fortran and C compilers on the workstation
  • Example and exercises codes from this lesson.
  • An interactive job run system is very desirable, i.e., not a batch queue job submission system.
