By the end of this lesson, students will be able to:

  • Define ARGB.
  • Know what a .png file is and the ARGB format, how it works, and its use cases.
  • Know how to convert an array of floats into scaled integers.
  • Understand the file formats in Fortran and C.
  • Know how to construct an alpha, red, green, blue 32-bit pixel.
  • Know by example how to avoid bad practices in Fortran/C programming that could lead to potential performance issues in data representations.
  • Draw on hands-on examples and exercises for building scientific applications with animations.
