The instructor of this lesson should possess the following abilities:

  • Be proficient in the Unix/Linux environment
  • Have a clear command of programming in basic programming languages such as C/C++/Java/ python and more if needed.
  • Be able to edit, compile, run and debug source codes written in the languages listed above
  • Be Familiar with parallel programming API such as MPI, OpenMP, CUDA, etc..
  • Be able to edit, compile, run, and debug code written in the above listed parallel programming API
  • Mostly be able to use a supercomputer workload manager and scheduler such as SLURM, TORQUE or any other well known workload managers beside these 2.
