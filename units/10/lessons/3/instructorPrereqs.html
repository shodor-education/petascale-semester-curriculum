Before starting the students should have:

  1. Basic familiarity with using a computer
  2. Basic familiarity with UNIX/LINUX environment
  3. Familiarity submitting jobs to Stampede2 cluster (module 3.9).
  4. Access to Stampede2.
  5. Familiarity with Visual Molecular Dynamics (VMD).
  6. XSEDE education allocation; takes 1- to 2- weeks to secure. Recommended usage is 500SUs per student. https://portal.xsede.org/allocations/education
