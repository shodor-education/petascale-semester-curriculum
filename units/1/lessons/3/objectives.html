By the end of this lesson, students will:

  1. By the end of the unit a student should be able to identify the differences between writing a program on a local and a remote machine.
  2. The student should be able to describe the differences between running a program locally and remotely.
  3. The student should be able to describe the use of a queueing system.
  4. The student should be able to describe the steps required to connect a local computer to a remote supercomputer to run and submit jobs remotely.
  5. The student should be able to describe the steps required to transfer data from/to a local computer to/from a remote supercomputer.
