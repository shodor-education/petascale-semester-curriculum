By the end of this lesson, students will be able to:

  • Identify the differences between writing a program on a local vs. a remote machine.
  • Describe the differences between running a program locally and remotely.
  • Describe the use of a queueing system.
  • Describe the steps required to connect a local computer to a remote supercomputer to run and submit jobs remotely.
  • Describe the steps required to transfer data from/to a local computer to/from a remote supercomputer.
