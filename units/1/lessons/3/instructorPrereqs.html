Before teaching this lesson, the instructor should have:

  • Access to a supercomputer or high performance computing environment and be able to provide access for the students taking the course. The instructor is advised to seek XSEDE Educational allocations. The instructor should include contact information that the students can reach out to for login issues, issues with running jobs/accessing the supercomputer, maintenance notifications, etc.
  • The ability to show the students how to run transfer code and data from and to the supercomputing resource.
  • The ability to show how to run a basic example (and be able to show the students what modules, if any, should be loaded to run a basic program) and how to request a compute node (PBS, SGE, etc).
  • Understanding of the differences between interactive and batch jobs.
