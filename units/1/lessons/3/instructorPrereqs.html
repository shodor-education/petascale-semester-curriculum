Prior to teaching this lesson, the instructor should understand and be able to do the following:

  1. 1. Have access to a supercomputer or high performance computing environment and be able to provide access for the students taking the course. The instructor is advised to seek XSEDE Educational allocations. The instructor should include contact information that the students can reach out to for login issues, issues with running jobs/accessing the supercomputer, maintenance notifications, etc.
  2. The instructor should be able to show the students how to run transfer code and data from and to the supercomputing resource.
  3. The instructor should be able to show how to run a basic example (and be able to show the students what lessons, if any, should be loaded to run a basic program) and how to request a compute node (PBS, SGE, etc).
  4. The instructor should understand the differences between interactive and batch jobs.
