By the end of this unit, students will be able to:

  • Know the difference between single-core computing and multicore computing.
  • Describe parallel programming and performance modeling basics that aid in executing applications to given specifications, terminate normally.
  • Understand how to compile, run, and submit a program to a queueing system.
  • Recall an application of HPC to a real-world scientific problem.
  • Understand the concepts behind why parallel programming can be useful and when it should be used.
